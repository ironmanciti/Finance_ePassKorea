# AI활용 데이터투자 분석 및 경제지표 자동화 과정

## 개요
Python 기반의 금융 데이터 분석 및 AI 활용 투자 분석 실습 교재입니다.
총 40차시로 구성되어 있으며, Google Colab 환경에서 실행됩니다.

## 커리큘럼 구성

### Module 01: 데이터 분석 환경 구축과 기초 (1~10차시)
Python 기반의 데이터 분석 환경을 구축하고, Pandas를 활용해 데이터를 다루는 기본기를 익힙니다.

| 차시 | 주제 | 구분 |
|------|------|------|
| 1 | Python 금융 데이터 분석 환경 세팅 | 이론/실습 |
| 2 | Python 기초 문법 (데이터 타입, 제어문) | 이론/실습 |
| 3 | Pandas 라이브러리 (Series, DataFrame) | 이론/실습 |
| 4 | Pandas로 금융 데이터 다루기 | 실습 |
| 5 | 데이터 전처리, 결측치 및 이상치 처리 | 이론/실습 |
| 6 | 데이터 시각화, Matplotlib과 Seaborn 기초 | 이론/실습 |
| 7 | 금융 시계열 데이터 시각화 (주가 차트 그리기) | 실습 |
| 8 | 국내 주식 데이터 수집 및 기본 분석 | 실습 |
| 9 | 이동평균을 이용한 금융 시계열 추세 분석 | 이론/실습 |
| 10 | 포트폴리오와 최적화 | 이론/실습 |
